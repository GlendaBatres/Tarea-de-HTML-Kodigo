ESTRUCTURAS DE CONTROL

If/Else Switch For While Do While

Estructura IF / ELSE

Se utiliza para tomar decisiones lógicas basadas en una condición verdadera o falsa.

if (puntos >= 6) {
    console.log("Aprobado");
} else {
    console.log("Reprobado");
}

Estructura SWITCH

Ideal para múltiples condiciones basadas en un solo valor.

switch (opcion) {
    case 1: abrirMenu(); break;
    default: salir();
}

Bucle FOR

Se utiliza cuando sabemos exactamente cuántas veces queremos repetir una acción.

for (let i = 0; i < 10; i++) {
    console.log("Iteración número: " + i);
}

Bucle WHILE

Repite el código mientras una condición sea verdadera.

while (energia > 0) {
    trabajar();
    energia--;
}

Bucle DO WHILE

Garantiza que el código se ejecute al menos una vez antes de verificar la condición.

do {
    preguntarNombre();
} while (nombre == "");
